# 🌱 CayCanhMNM - Web quản lý cây cảnh

## 📌 Giới thiệu
Dự án web quản lý và tìm kiếm cây cảnh.

## 🛠 Công nghệ sử dụng
- Frontend: ReactJS
- Backend: (ghi rõ: Node / Django / Spring Boot)
- Database: PostgreSQL

## 📁 Cấu trúc thư mục

CayCanhMNM/
│
├── backend/ # Backend API
├── frontend/ # React frontend
└── README.md


## 🚀 Cách chạy project

### 1. Backend

cd backend

chạy tùy theo công nghệ

### 2. Frontend

cd frontend
npm install
npm start


## 🌐 API

http://localhost:8080/api/
